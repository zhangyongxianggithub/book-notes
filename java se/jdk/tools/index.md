Java Development Kit Version 21 Tool Specifications
All Platfroms
- jar, 创建一个classes/resources的归档文件，或者操作/重新存储归档文件中单独的class/resource
- jarsigner, 签名或者校验Java Archive文件
- java, 启动一个Java应用
- javac, 读取.java文件，把它们编译为字节码类文件
- javadoc, 从Java源代码文件中生成HTML形式的API文档
- javap, 反编译一个或者多个class文件
- jcmd, 向一个运行中的JVM发送诊断命令请求
- jconsole, 启动一个图形化的console来monitor/manage Java应用
- jdb, 查询并修复Java平台programs中的bugs
- jdeprscan, 一个静态分析工具，扫描一个jar文件，找出使用了deprecated API地方
- jdeps, 启动Java类依赖分析器，Java class dependency analyzer
- jfr, 解析并打印Flight Recorder文件
- jhsdb, attach to a Java process or launch a postmortem debugger to analyze the content of a core dump from a crashed Java Virtual Machine (JVM)
- jinfo, 生成一个特定Java进程的Java配置信息
- jlink, 组装并优化一组模块与他们的依赖到一个自定义的runtime image
- jmap, 打印一个特定JVM进程的详细信息
- jmod, 创建一个JMOD文件并且列出已经存在的JMOD文件的内容
- jpackage, package一个self-contained Java应用
- jps, 列出目标系统上的JVMs
- jrunscript, 运行一个command-line脚本shell，shell支持interactive与batch模式
- jshell, 交互式的对声明、语句、表达式求值，在一个read-eval-print loop中
- jstack, 打印一个特定的Java进程的Java线程的stack traces
- jstat, monitor JVM statistics
- jstatd, monitor the creation and termination of instrumented Java HotSpot VMs
- jwebserver, 启动一个Java的简单的web server
- keytool, manage a keystore (database) of cryptographic keys, X.509 certificate chains, and trusted certificates
- rmiregistry, create and start a remote object registry on the specified port on the current host
- serialver, return the `serialVersionUID` for one or more classes in a form suitable for copying into an evolving class
